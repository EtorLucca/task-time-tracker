# Task Tracker

Visualizar o projeto: <a href="https://task-time-tracker.vercel.app/">na Vercel</a>

## Como rodar

### Clonar o projeto:
```
git clone https://github.com/EtorLucca/task-time-tracker.git
```

### Instalar as dependências:

```
yarn install
```

### Rodar o projeto na porta padrão:
```
yarn serve
```
